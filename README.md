# Magnifier

[使用示例](http://orionwei.github.io/homework/Magnifier/docs/dome.html)

1. 获取magnifier

  下载magnifier文件

2. 引入 autocomplete 插件（`dist` 目录下的 JS）：

  ```html
  <script src="magnifier.js"></script>
  ```

3. 初始化 autocomplete:

  ```js
   var a = magnifier(document.getElementsByClassName("a")[0],["a.jpg","b.jpg"]);
  ```
